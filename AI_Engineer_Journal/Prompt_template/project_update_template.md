# 📊 Project Update & Learning Journal Template

## 🎯 **ПРОМПТ ДЛЯ AI АГЕНТА**

```markdown
Ты технический writer и career advisor для AI Engineer. 
Помоги мне структурированно задокументировать обновления моего проекта и обучение.

Проанализируй предоставленную информацию и создай профессиональную запись для моего портфолио, используя template ниже.

**Фокус на:**
- Конкретные технические решения и их обоснование
- Измеримые улучшения (performance, код, данные)
- Новые навыки и технологии
- Проблемы и их решения
- Готовность к собеседованию (что можно рассказать)

**Стиль:** Технический, конкретный, с метриками, готовый для резюме/интервью
```

## 💡 **PRO TIPS**

### **Для качественных записей:**
1. **Всегда включайте цифры** - время, производительность, размер данных
2. **Документируйте reasoning** - почему выбрали именно это решение
3. **Сохраняйте code snippets** - ключевые части реализации
4. **Записывайте ошибки** - они становятся лучшими историями для интервью
5. **Отслеживайте progression** - как навыки развиваются со временем

### **Для интервью:**
1. **Готовьте STAR stories** - Situation, Task, Action, Result
2. **Количественные результаты** - "улучшил производительность на 50%"
3. **Technical depth** - готовьтесь к глубоким техническим вопросам
4. **Lessons learned** - что бы сделали по-другому
5. **Future thinking** - как планируете развивать проект
