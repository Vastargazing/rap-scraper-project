# 🦙 Установка и настройка Ollama для AI анализа

## 📋 Что такое Ollama?

Ollama - это бесплатная локальная платформа для запуска LLM моделей на вашем компьютере. Альтернатива дорогим API вроде GPT-4 и Gemini.

### ✅ Преимущества Ollama:
- 🆓 **Полностью бесплатно** (в отличие от Gemini 50 запросов/день)
- 🔒 **Приватность** - данные не покидают ваш компьютер
- ⚡ **Быстрая работа** после установки
- 🎯 **Высокое качество** анализа с моделями llama3.1

---

## 🛠️ Установка Ollama

### Windows:
1. Скачайте с [ollama.com](https://ollama.com/download)
2. Запустите `OllamaSetup.exe`
3. Ollama автоматически запустится как сервис

### macOS:
```bash
# Через Homebrew
brew install ollama

# Или скачайте .dmg с сайта
```

### Linux:
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Проверка установки:
```bash
ollama --version
# Должно показать версию, например: ollama version is 0.1.32
```

## 🤖 Установка моделей

### Рекомендуемые модели для анализа текстов:

```bash
# Основная модель (8B параметров, ~4.7GB)
ollama pull llama3.1:8b

# Альтернативы:
ollama pull llama3.1:7b      # Меньше, но быстрее
ollama pull codellama:13b    # Лучше для структурированного вывода
ollama pull mistral:7b       # Хорошее качество/скорость
```

### Проверка установленных моделей:
```bash
ollama list
```

## 🚀 Запуск и тестирование

### Запуск Ollama (если не запущен автоматически):
```bash
ollama serve
```

### Тест в командной строке:
```bash
ollama run llama3.1:8b "Analyze this rap lyrics: 'Started from the bottom now we here'"
```

### Тест через API:
```bash
curl http://localhost:11434/api/generate -d '{
  "model": "llama3.1:8b",
  "prompt": "Hello world",
  "stream": false
}'
```

## ⚙️ Настройка производительности

### Параметры модели в Ollama:
```python
# В multi_model_analyzer.py можно настроить:
{
    "model": "llama3.1:8b",
    "options": {
        "temperature": 0.1,    # Низкая = более детерминированный
        "top_p": 0.9,         # Фокус на лучших токенах
        "max_tokens": 1500,   # Длина ответа
        "num_ctx": 4096       # Размер контекста
    }
}
```

### Мониторинг ресурсов:
- **RAM:** ~8GB для llama3.1:8b
- **Диск:** ~4.7GB на модель
- **CPU/GPU:** Ollama использует доступные ресурсы

## 🔄 Fallback на DeepSeek

Если Ollama недоступен, система автоматически переключится на DeepSeek API.

### Настройка DeepSeek:
1. Регистрация на [platform.deepseek.com](https://platform.deepseek.com)
2. Получение API ключа
3. Добавление в `.env`:
```bash
DEEPSEEK_API_KEY=your_api_key_here
```

### Стоимость DeepSeek:
- **Input tokens:** $0.56 / 1M токенов
- **Output tokens:** $1.68 / 1M токенов
- **Примерно:** $0.003 за анализ одной песни

## 🐛 Troubleshooting

### Ollama не запускается:
```bash
# Проверить процесс
ps aux | grep ollama

# Перезапустить
ollama serve

# Проверить порт
netstat -an | grep 11434
```

### Модель загружается медленно:
- Убедитесь в стабильном интернете
- Проверьте свободное место на диске (>5GB)

### Ошибки анализа:
- Проверьте логи: `tail -f ai_analysis.log`
- Убедитесь в корректности JSON ответа
- Попробуйте другую модель: `ollama pull mistral:7b`
