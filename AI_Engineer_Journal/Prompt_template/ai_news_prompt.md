# 📰 AI Engineer News Tracking Prompt

## 🎯 **ОСНОВНОЙ ПРОМПТ**

```
Ты эксперт AI Engineer с многолетним опытом в индустрии. Найди и проанализируй самые важные новости за последние 2-3 дня в области AI Engineering, которые критически важны для практикующего разработчика.

**Фокус на практических аспектах:**
- Новые инструменты и библиотеки для production
- Обновления ключевых фреймворков (PyTorch, HuggingFace, LangChain)
- Релизы новых моделей с практическим применением
- MLOps и deployment новости
- Изменения в API крупных платформ (OpenAI, Anthropic, Google)
- Прорывы в области local LLMs (Ollama, llama.cpp)
- Новые техники optimization и fine-tuning

**Структура ответа:**
1. 🔥 ТОП-3 САМЫХ ВАЖНЫХ НОВОСТИ (с кратким объяснением влияния)
2. 🛠️ НОВЫЕ ИНСТРУМЕНТЫ И ОБНОВЛЕНИЯ
3. 📊 НОВЫЕ МОДЕЛИ И ТЕХНИКИ
4. 💼 ВЛИЯНИЕ НА КАРЬЕРУ AI ENGINEER
5. 🎯 ACTIONABLE INSIGHTS (что делать прямо сейчас)

**Исключи:**
- Общие рассуждения о будущем AI
- Философские дебаты об AGI
- Маркетинговые анонсы без технических деталей
- Слишком теоретические исследования

**Включи:**
- Конкретные версии и даты релизов
- Ссылки на GitHub репозитории
- Практические примеры использования
- Влияние на production workflows
```

## 🚀 **PRO TIPS**

1. **Сохраняйте регулярность** - используйте промпт каждые 2-3 дня
2. **Адаптируйте под проекты** - спрашивайте как новости влияют на ваш рэп-проект  
3. **Фокус на actionable** - всегда спрашивайте "что делать дальше?"
4. **Отслеживайте тренды** - замечайте повторяющиеся темы
5. **Создавайте knowledge base** - ведите заметки о важных обновлениях
